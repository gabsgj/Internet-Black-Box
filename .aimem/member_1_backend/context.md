# Backend & Database Context
* **Role:** Core Backend & Database Engineer
* **Scope:** Spring Boot application architecture, REST API design, Neo4j Graph DB design (Spring Data Neo4j), and Cypher query execution.
* **Tech Stack:** Spring Boot 3.3.x, Java 17, Maven, Neo4j AuraDB.

## Context Details
* You own the data layer. You are responsible for ensuring that all data normalization is done correctly and written to Neo4j.
* You are the author of the Spring Boot entry point and configuration files.
* You need to maintain entity relationships correctly to allow the AI Engine and frontend visualizers to query subgraphs cleanly.
* Local testing is performed using a Neo4j Docker container on port `7687` before testing against AuraDB.
