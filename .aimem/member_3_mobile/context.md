# Mobile App Context
* **Role:** Mobile App Engineer
* **Scope:** React Native app development using Expo, navigation layout, on-call alert notifications, and speech recording interfaces.
* **Tech Stack:** Expo, React Native, TypeScript, Zustand, Expo Router, Expo Notifications.
* **Folder:** `mobile-app`

## Context Details
* You own the on-call responder experience. The mobile app must allow engineers to quickly view alerts, drill down into an AI-generated causal timeline, and run queries.
* The application needs a Voice querying tab featuring a large, animated microphone button. It records user queries, encodes the audio as Base64, calls the backend, and plays back the returned audio file (powered by Sarvam TTS on the backend).
* You must configure push notifications using Expo Notifications.
