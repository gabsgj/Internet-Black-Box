# Frontend Dashboard Context
* **Role:** Frontend Dashboard Engineer
* **Scope:** Single Page Web Application (React, Vite, TypeScript), responsive layout, data visualizations, and interactive causal graph rendering.
* **Tech Stack:** React 18, Vite, TypeScript, Tailwind CSS, Sigma.js, Recharts.
* **Folder:** `web-dashboard`

## Context Details
* You own the user-facing visual experience. The dashboard must look premium and high-end, utilizing dark mode themes, subtle gradient transitions, and responsive cards.
* The critical piece of your deliverable is the interactive Causal Graph Visualization. It must map the JSON subgraphs from the backend (`/api/incidents/{id}/graph`) into nodes and relationships.
* You need to connect to WebSocket channels to enable real-time dashboard updates when new incidents or events are ingested.
