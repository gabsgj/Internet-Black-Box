Neo4j Causal Graph Database

Querying Lineage, Not Tables

By modeling developers, code diffs, deployments, chats, and alerts as connected nodes, Internet Black Box converts diagnostic queries into simple, constant-time graph traversals.

  • Index-Free Adjacency: Traces lineage without expensive SQL JOINs.
  • Causal Traversal: Connects nodes via :AUTHORED, :DEPLOYED_TO, :AFFECTED, and :CAUSED_BY edges.
  • Root Cause Extraction: Computes the shortest path from an incident to the originating commit.
// Neo4j Causal Path Tracing
MATCH (i:Incident {id: $id})
MATCH path = shortestPath((c:Event {type:'COMMIT'})-[:TRIGGERED|PRECEDED*]->(i))
RETURN path

Render Workflows Integration

Reliable, Multi-Stage Ingestion

01. `ingest-github`
Scheduled Cron Ingestion

Scheduled every 5 minutes. Pulls new commits, PR approvals, and deployment triggers into AuraDB. Normalizes diff files asynchronously.

02. `ingest-slack`
Real-Time Event Stream

Event-driven webhook listener. Captures channels and filters messages. Extracts metadata to generate direct reference edges.

03. `reconstruct-incident`
9-Stage AI Worker

On-demand background worker. Queries the 4-hour temporal subgraph, compiles variables, and calls NVIDIA NIM for final timeline reconstruction.
